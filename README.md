# Go little scraper

À n'utiliser que dans un but éducatif.

## Installation

Installer [Go](https://go.dev/doc/install)

Installer les packages

```bash
  go install
```

## Utilisation

- Remplacer l'url du site dans `main.go`
- Éxecuter le code

```bash
  go run main.go
```

- Une fois terminé le fichier `fr.csv` contiendra les résultats
